This tool is built for people who want a faster local setup with fewer parts to install. You download one file, open it, and start working.

## 🧭 Troubleshooting

### The file does not open

- Check that the download finished
- Make sure you extracted the ZIP file if one was downloaded
- Try running the file again
- Right-click the file and choose Run as administrator if needed

### Windows shows a security prompt

- Choose More info if needed
- Select Run anyway if you trust the download source
- Confirm that you downloaded the latest release from the release page

### The window opens and closes fast

- Open the app from a terminal window if the release includes a CLI build
- Check whether the file was placed in a folder with full access
- Try downloading the file again

### The app cannot connect to AI tools

- Check your internet connection
- Confirm your API or service settings
- Make sure the session config is correct
- Restart the app and try again
